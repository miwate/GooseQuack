# GooseQuack

## Préalable : Gradle (sur Ubuntu)
```bash
sudo apt install gradle
```
Versions :
Java 21
Gradle 8.5

## Comment build/run
Sur linux (liste non exhaustive de commandes)
```bash
./gradlew build
```
```bash
./gradlew run
```
```bash
./gradlew clean
```
```bash
./gradlew test
```

Sur Windows idem mais ``gradlew.bat``

## Générer la javadoc
```bash
./gradlew javadoc
```
